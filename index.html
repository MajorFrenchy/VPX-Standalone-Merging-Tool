What is it?

The VPX Standalone Merging Tool is a powerful desktop application designed for Virtual Pinball enthusiasts. It automates the tedious process of organizing and merging all the components needed for a complete VPX table setup. Simply drag and drop your .vpx files, and let the tool handle the rest!

    Download VPXmerge.py from the Releases page
  4. Run the application
    Double-click VPXmerge.py or run from Command Prompt:

üçé macOS Installation

  1. Install Python 3.8 or higher
    Download from python.org
    Or use Homebrew: brew install python3
  2. Install Tkinter (if not already installed)
    Open Terminal and run: 
    brew install python-tk@3.12
    Note: Adjust version number to match your Python version
  3. Install required packages
    In Terminal, run: 
    pip3 install tkinterdnd2 olefile
  4. Download the script
    Download VPXmerge.py from the Releases page
  5. Run the application
    In Terminal, navigate to the download folder and run: 
    python3 VPXmerge.py
